# CICDtestrepo

A simple Express.js API built with TypeScript, designed for testing CI/CD pipelines and automation workflows.

## 🚀 Features

- **Express.js API Server** - Lightweight HTTP server for testing deployments
- **TypeScript Support** - Full type safety with TypeScript
- **Docker Ready** - Includes Dockerfile and Docker Compose configuration
- **Development Tools** - Nodemon for hot reloading, tsx for TypeScript execution
- **Auto-Deploy Testing** - Perfect for testing continuous integration and deployment workflows

## 📋 Prerequisites

- **Node.js** 18+ or Docker
- **npm** 9+ (if running locally)
- **Docker** (optional, for containerized deployment)
